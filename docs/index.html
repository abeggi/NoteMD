Features

Everything you need to manage your knowledge base

✍

Live Preview

Write in markdown with real-time WYSIWYG preview. Syntax highlighting, tables, and task lists built in.

🔒

AES-256 Encryption

Documents encrypted at rest. Your data stays private even on shared infrastructure.

👥

Multi-User & Multi-Org

Create accounts for your team with role-based access. Keep workspaces separate with organizations.

🔍

Full-Text Search

Search across all documents instantly with FTS5. Find anything in seconds.

📎

File Attachments

Upload images, PDFs, and other files. Embed them directly in your documents.

📈

Mermaid Diagrams & Math

Render flowcharts, diagrams, and LaTeX math inline. Perfect for technical notes.

Quick Setup

Get up and running in minutes with Docker

docker run -d \
  --name notemd \
  -p 3000:3000 \
  -v ./data:/data \
  abeggi/notemd:latest

Then open http://localhost:3000
